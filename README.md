# Pawan Portfolio - GitHub Pages Ready

This project is ready for GitHub Pages deployment using GitHub Actions.

## Before you upload
This ZIP is configured for a GitHub repository named:

`pawan-portfolio`

If your repository name is different, update this line in `vite.config.js`:

```js
base: '/pawan-portfolio/',
```

Replace `pawan-portfolio` with your exact repo name.

## Upload steps
1. Create a new GitHub repository.
2. Upload all files from this ZIP.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, set **Source = GitHub Actions**.
5. Push to the `main` branch.

Your site will publish automatically.
